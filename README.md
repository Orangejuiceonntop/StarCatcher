# Star-Catcher

A WarioWare styled pixel game where your main goal is to collect the stars in each minigame!

Try here: https://orangejuiceontop.itch.io/star-catcher

- 3 unique minigames
- Special win and lose screens
- Custom pixel assets and sprites
